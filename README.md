# 📊 WhatsApp Chat Analyzer  

## 🚀 Overview  
WhatsApp Chat Analyzer is a **Streamlit** web app that analyzes your WhatsApp chat data. It provides insights into user activity, message trends, media sharing, emojis, and word usage.  

## 🔥 Features  
✅ Upload and analyze WhatsApp chat exports  
✅ View message count, word count, media, and links shared  
✅ Visualize **monthly & daily message trends**  
✅ Activity heatmap for chat engagement analysis  
✅ Identify the **most active users** in a group chat  
✅ Generate a **word cloud** to highlight frequently used words  
✅ Emoji usage statistics with pie charts  
✅ Common word frequency analysis  

## 🛠️ Tech Stack  
- **Frontend:** Streamlit  
- **Backend:** Python, Pandas  
- **Visualization:** Matplotlib, Seaborn, WordCloud  
- **Natural Language Processing:** Emoji Analysis, WordCloud
